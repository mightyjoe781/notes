# Module 1 - First Chords

## D and A Chords

The first two chords most beginners learn. Practice transitioning between them cleanly.

- **D major**: fingers on strings 1, 2, 3 in the first position
- **A major**: barre or three-finger version across strings 2, 3, 4

## One Minute Changes

A core JustinGuitar exercise: set a timer for 1 minute and count how many clean chord changes you can make. Track your count each session - improvement is gradual but measurable.

## Perfect Chord Exercise

Hold each chord for 30-60 seconds, checking every string rings clearly. Adjust finger placement until each note is clean before moving on.
